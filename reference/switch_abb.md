# Convert an abbreviated month name to a numeric value

Convert an abbreviated month name to a numeric value

## Usage

``` r
switch_abb(month)
```

## Arguments

- month:

  a character string specifying one of the twelve months, i.e. ("Jan",
  "Feb", "Mar", "Apr", "May", "Jun", "Jul", "Aug", "Sep", "Oct", "Nov",
  "Dec")

## Value

a character string which will correspond to the numeric value of the
input month, i.e. ("01", "02", "03", "04", "05", "06", "07", "08", "09",
"10", "11", "12")
